# 📱 ContactosApp

**ContactosApp** es una aplicación para la gestión de contactos.  
Permite registrar, buscar, editar y eliminar contactos de manera sencilla y eficiente.

---

## 🚀 Características principales

- 📇 Gestión completa de contactos (crear, listar, editar, eliminar).  
- 🔍 Búsqueda rápida por nombre o número.  

---

## 🛠️ Tecnologías utilizadas

- **Lenguaje:** Java  
- **Control de versiones:** Git + GitHub
